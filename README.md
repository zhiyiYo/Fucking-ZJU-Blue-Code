# Fucking-ZJU-Blue-Code
折姜大学蓝码生成器

## 快速开始
1. 搭建环境

   ```shell
   pip install -r requirements.txt
   ```
2. 配置用户名和密码，创建 `config/config.json`，在里面写入：
   ```json
    {
        "username": "学号",
        "password": "密码",
        "nickname": "一级棒卡哇伊的硝子酱(可选)"
    }
   ```

3. 启动服务器
   ```shell
   python app.py
   ```
